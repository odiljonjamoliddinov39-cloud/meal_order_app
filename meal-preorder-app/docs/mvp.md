# MVP Scope

## Customer
- Open Telegram bot
- Launch Mini App
- View upcoming menu days
- Browse meals by date
- Add meals to cart
- Place preorder before deadline
- View own orders

## Admin
- Login
- Create future menu days
- Add menu items with quantity, price, image
- Close/open order day
- View all orders by date
- See sold counts and remaining stock

## Backend
- Telegram user identity handling
- Stock-safe checkout transaction
- CRUD for menu days and items
- Order creation and listing
- Admin login scaffold
